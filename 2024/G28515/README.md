# 課題 『ビジネスアイデア』

### 興味を持っているビジネス領域・業界

- 娯楽・レジャー（ゲーム）

## 上記のビジネス領域で注目する課題

- 少子高齢化によって、若者のプレイ人口が減少してしまう
- VR・ARを利用したゲームが子供に悪影響を及ぼしてしまう
- eスポーツの競技環境が整っていない

## 上記のビジネス領域で提案する解決策

- 高齢者向けに脳トレや運動を促進するVR・ARゲームを開発する
- VR酔いを防ぐために動きに応じて視覚効果を調整する、視力に配慮した設計を行う、強制的にゲームを終了する機能を導入する
- eスポーツ協会のような全体を統括する組織を設立する
